siempre estare aqui para ti

Cariño, siempre estaré aquí para ti
No dije que la vida es fácil.
Tiene sus propios altibajos.
Hay momentos de tristeza y momentos de felicidad

cuando estas molesto
Cuando necesitas un hombro en el que apoyarte
Cuando tienes ganas de llorar
siempre estare aqui para ti

Durante tiempos difíciles
estaré ahí para consolarte
Durante los momentos felices
Estaré allí para disfrutar cada momento contigo.

Siempre puedes contar conmigo
Cuando las cosas no parecen funcionar
Siempre puedes llamarme
siempre estare aqui para ti

Si necesitas a alguien con quien hablar
siempre estare aqui para ti
mi tiempo será todo tuyo
Mi corazón, mis oídos y mi mente estarán listos para escuchar

Si necesitas a alguien con quien reír
Si necesitas a alguien que te haga sonreír
Si necesitas a alguien con quien dar un paseo
Siempre estare aqui para ti.
